# 数据科学导论 (DS101)

## 课程简介
本课程是数据科学辅修的入门课程，介绍数据科学的基本概念、工作流程和常用工具。
学生将通过实际案例学习 Python 数据处理、可视化和基础统计分析，建立数据思维。

## 学习目标
- 理解数据科学的工作流程（数据采集 → 清洗 → 分析 → 可视化）
- 掌握 Python 基础语法与常用数据分析库（Pandas、NumPy）
- 掌握数据可视化方法（Matplotlib、Seaborn）
- 理解描述性统计与基础推断统计
- 能够独立完成小型数据分析项目

## 课程大纲

### 第一部分：数据科学概述（第 1-2 周）
- 数据科学的定义与应用领域
- 数据分析的基本流程
- Python 环境搭建（Anaconda + Jupyter）

### 第二部分：Python 编程基础（第 3-6 周）
- Python 基本语法与数据结构
- NumPy：数组操作与数值计算
- Pandas：DataFrame 操作与数据清洗
- 实战：清洗一个真实数据集

### 第三部分：数据可视化（第 7-10 周）
- Matplotlib 基础绘图
- Seaborn 统计可视化
- 交互式可视化简介（Plotly）
- 实战：制作一份数据分析报告

### 第四部分：统计分析入门（第 11-16 周）
- 描述性统计：均值、方差、分位数
- 概率分布：正态分布、二项分布
- 假设检验基础
- 相关分析与回归分析入门
- 期末项目：完整的数据分析报告

## 推荐教材
1. 《利用 Python 进行数据分析》（第 2 版），Wes McKinney 著
2. 《统计学习方法》，李航著（入门章节）

## 学习建议
- 每周投入约 5-7 小时课外学习（含编程练习）
- 重视动手实践，每节课后务必运行示例代码
- Pandas 是数据分析的核心工具，需要反复练习
- 期中/期末项目建议选取自己感兴趣的数据集
