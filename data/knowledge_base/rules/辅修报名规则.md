# 本校辅修报名规则

## 报名条件
1. 全日制本科二年级及以上学生
2. 主修专业已修课程全部及格
3. 平均学分绩点（GPA）不低于 2.0
4. 每名学生限报 1 个辅修专业

## 报名时间
- **秋季学期**：第 1-2 周提交申请 → 第 3 周公布审核结果 → 第 4 周开始上课
- **春季学期**：第 1-2 周提交申请 → 第 3 周公布审核结果 → 第 4 周开始上课

## 选课规则
1. 每学期限选 2 门辅修课程（6 学分上限）
2. 必须按课程体系顺序选修（先修未过，不能选后续课程）
3. 辅修课程挂科可补考 1 次，补考未过须重修
4. 辅修课程可申请免修（需提供同等课程成绩证明）

## 学分要求
- 辅修总学分 ≥ 12 且核心课全部通过 → 颁发辅修证书
- 辅修总学分 ≥ 8 但 < 12 → 颁发辅修课程证明（非学位）
- 辅修总学分 < 8 → 无证明

## 退选规则
1. 开课后 2 周内可无理由退选（全额退费）
2. 开课第 3-8 周退选 → 成绩单标注"退选"，不退费
3. 第 8 周后不可退选

## 冲突处理
1. 辅修与主修上课时间冲突 → 优先保障主修，可申请辅修换班
2. 辅修与主修考试时间冲突 → 可申请辅修课程缓考
3. 两个辅修课程之间冲突 → 建议调整选课计划
4. 换班/缓考申请需在第 3 周前提交至开课学院教务办

## 联系方式
- 教务处辅修管理科：主校区行政楼 302
- 各学院教务办联系方式见学院官网
