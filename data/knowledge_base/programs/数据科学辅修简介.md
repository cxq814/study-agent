# 数据科学辅修专业简介

## 培养目标
培养掌握数据处理、统计分析和机器学习基本方法，能够从海量数据中提取有价值信息的跨学科人才。

## 核心能力
- 熟练使用 Python 进行数据处理与分析
- 掌握统计学基本概念与推断方法
- 理解经典机器学习算法的原理与应用
- 能够独立完成数据可视化与报告撰写

## 课程体系

## 辅修要求
1. 主修专业为计算机、数学、统计、经管类优先
2. 已修完高等数学、线性代数
3. 具备基础编程能力（Python）
4. 辅修总学分 ≥ 12 且核心课全部通过

## 报名流程
同金融辅修，详见教务处通知。

## 常见问题
- **Q: 需要很好的编程基础吗？**
  A: 入门课程会从 Python 基础教起，但具备编程经验更容易跟上进度。
- **Q: 数学要求高吗？**
  A: 机器学习课程需要一定的线性代数和概率论基础，建议提前复习相关知识点。
