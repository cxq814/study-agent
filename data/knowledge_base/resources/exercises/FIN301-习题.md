# 投资学 课后习题

## 第一章：资产组合理论

### 计算题
1. 股票A期望收益12%，标准差20%；股票B期望收益8%，标准差12%；相关系数0.3。若等权重投资，求组合的期望收益和标准差。
   **解答**：Rp = 0.5×12% + 0.5×8% = 10%
   σp² = 0.5²×0.04 + 0.5²×0.0144 + 2×0.5×0.5×0.3×0.2×0.12 = 0.01+0.0036+0.0036 = 0.0172
   σp = √0.0172 = 13.11%

2. 无风险利率3%，市场组合期望收益10%，标准差15%。某股票β=1.5，求其CAPM期望收益。
   **解答**：E(R) = 3% + 1.5×(10%-3%) = 3% + 10.5% = 13.5%

3. 股票β=0.8，α=2%（年化），市场收益12%。如果CAPM成立，实际收益应是多少？
   **解答**：CAPM预测：Rf + 0.8×(12%-Rf)，若无Rf=3%，则预测=3%+7.2%=10.2%。α=2%表示实际=12.2%，被低估。

## 第二章：期权定价

### 计算题
1. 使用Black-Scholes公式相关概念：当前股价50元，行权价52元，无风险利率3%，波动率30%，期限6个月。判断看涨期权是否为实值？
   **解答**：S=50 < K=52，看涨期权为虚值（out-of-the-money）。

2. 一个看跌期权的行权价100元，当前股价95元，期权费8元。求买方和卖方的盈亏平衡点。
   **解答**：买方盈亏平衡 = 100-8 = 92元（股价低于92元盈利）；卖方盈亏平衡同上。

## 第三章：行为金融

### 简答题
1. 什么是"处置效应"（Disposition Effect）？
   **要点**：投资者倾向于过早卖出盈利股票、过久持有亏损股票。

2. 列举三个常见的投资心理偏差并举例说明。
   **要点**：过度自信（频繁交易）、锚定效应（固守买入价）、羊群效应（追涨杀跌）。
