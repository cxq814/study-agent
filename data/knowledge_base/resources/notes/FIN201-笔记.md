# 公司金融 核心知识点笔记

## 一、资本预算

### 1.1 核心评估方法
| 方法 | 公式 | 决策规则 |
|------|------|----------|
| NPV | ΣCFᵼ/(1+r)ᵼ - I₀ | NPV > 0 接受 |
| IRR | NPV = 0 时的 r | IRR > 要求回报率 接受 |
| 回收期 | 累计回本时间 | < 基准回收期 接受 |
| PI | (NPV + I₀)/I₀ | PI > 1 接受 |

### 1.2 NPV与IRR比较
- NPV 始终正确（价值可加性）
- IRR 在非标准现金流时可能多解
- 互斥项目必须用NPV

### 1.3 现金流估算
- 增量现金流原则（不包含沉没成本）
- 考虑机会成本
- 考虑营运资本变动
- 不考虑融资成本（已在折现率体现）

## 二、资本结构

### 2.1 MM定理
- **MM I（无税）**：企业价值与资本结构无关
- **MM I（有税）**：Vₗ = Vᵤ + T×D（债务税盾）
- **MM II（有税）**：Re = Ru + (Ru-Rd)×(1-T)×(D/E)

### 2.2 权衡理论
最优资本结构：税盾边际收益 = 财务困境边际成本

### 2.3 优序融资理论
内部融资 > 债务融资 > 股权融资

### 2.4 WACC 计算
WACC = E/V × Re + D/V × Rd × (1-T)

## 三、股利政策

### 3.1 股利无关论（MM）
在完美市场中，股利政策不影响企业价值

### 3.2 现实因素
- 税收：资本利得税率 < 股息税率 → 偏好回购
- 信号效应：增发股利 = 管理层对未来有信心
- 代理成本：高股利减少自由现金流，约束管理层

### 3.3 常见股利政策
- 剩余股利政策
- 稳定增长股利政策
- 固定支付率政策
- 低正常+额外股利

## 四、营运资本管理

- 现金周转期 = 存货周转期 + 应收账款周转期 - 应付账款周转期
- 最优现金持有量：Baumol模型（交易成本与机会成本平衡）
