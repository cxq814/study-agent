# 金融学基础 核心知识点笔记

## 一、货币与货币制度

### 1.1 货币的本质
- 货币是从商品中分离出来的固定充当一般等价物的特殊商品
- 体现了一定的社会生产关系

### 1.2 货币职能（五大职能）
| 职能 | 说明 | 示例 |
|------|------|------|
| 价值尺度 | 衡量商品价值 | 标价 |
| 流通手段 | 商品交换媒介 | 现金购物 |
| 贮藏手段 | 保存价值 | 储蓄 |
| 支付手段 | 清偿债务 | 还贷 |
| 世界货币 | 国际结算 | 美元储备 |

### 1.3 货币制度演变
金银复本位制 → 金本位制 → 信用货币制度

**核心概念：格雷欣法则** — 劣币驱逐良币

## 二、利率理论

### 2.1 利率决定理论
- **古典利率理论**：储蓄和投资决定利率
- **凯恩斯流动性偏好理论**：货币供给与需求决定利率
- **可贷资金理论**：综合上述两种理论

### 2.2 利率期限结构
- 预期理论：长期利率 = 未来短期利率预期的平均值
- 市场分割理论：不同期限市场各自独立
- 流动性溢价理论：长期利率 = 预期短期利率均值 + 流动性溢价

### 2.3 利率计算
- 单利：I = P × r × n
- 复利：FV = PV × (1 + r)^n
- 实际利率 ≈ 名义利率 - 通货膨胀率

## 三、金融体系

### 3.1 金融市场分类
- 按期限：货币市场 vs 资本市场
- 按发行：一级市场 vs 二级市场
- 按交易：场内市场 vs 场外市场

### 3.2 金融机构体系
中央银行 → 商业银行 → 非银行金融机构（证券、保险、基金等）

### 3.3 商业银行
- 资产业务：贷款、证券投资
- 负债业务：存款、借款
- 中间业务：结算、代理、咨询

**存款创造**：法定准备金率决定货币乘数上限
M1 = 基础货币 × 货币乘数

## 四、中央银行与货币政策

### 4.1 央行职能
发行的银行、银行的银行、政府的银行

### 4.2 货币政策工具
| 工具 | 说明 | 效果 |
|------|------|------|
| 法定准备金率 | 调整存款准备金比例 | 效果最猛 |
| 再贴现政策 | 调整对商业银行贷款利率 | 间接温和 |
| 公开市场操作 | 买卖有价证券 | 最常用 |

### 4.3 货币政策传导机制
政策工具 → 操作目标 → 中介目标 → 最终目标
