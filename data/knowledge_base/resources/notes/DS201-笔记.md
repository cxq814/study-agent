# 机器学习基础 核心知识点笔记

## 一、机器学习概述

### 三大范式
| 范式 | 特点 | 典型算法 |
|------|------|----------|
| 监督学习 | 有标签训练 | 回归、分类 |
| 无监督学习 | 无标签 | 聚类、降维 |
| 强化学习 | 奖励信号 | Q-Learning |

### 核心概念
- **偏差（Bias）**：模型预测与真实值的系统性偏离
- **方差（Variance）**：模型对训练数据变化的敏感度
- **偏差-方差权衡**：简单模型高偏差低方差，复杂模型低偏差高方差

## 二、经典算法

### 线性回归
y = β₀ + β₁x₁ + ... + βₚxₚ + ε
- 损失函数：MSE
- 求解：最小二乘法 / 梯度下降
- 正则化：Ridge(L2), Lasso(L1), ElasticNet

### 逻辑回归
P(y=1|x) = 1/(1 + e^{-βᵀx})
- 用于二分类
- 损失函数：交叉熵
- 输出可解释为概率

### 决策树
- ID3/C4.5：信息增益/增益率
- CART：Gini指数
- 剪枝：预剪枝 + 后剪枝
- 优点：可解释性强；缺点：容易过拟合

### 支持向量机（SVM）
- 核心思想：最大化分类间隔
- 核技巧：线性核、多项式核、RBF核
- 软间隔：引入松弛变量，参数C控制

### KNN
- 惰性学习，无显式训练
- 距离度量：欧氏距离、曼哈顿距离
- k值选择：小k高方差，大k高偏差

## 三、模型评估

### 分类指标
- 混淆矩阵：TP, FP, TN, FN
- 精确率 Precision = TP/(TP+FP)
- 召回率 Recall = TP/(TP+FN)
- F1 = 2PR/(P+R)
- ROC-AUC：越接近1越好

### 回归指标
- MAE, MSE, RMSE, R²

### 验证方法
- Holdout：训练/测试分割
- K折交叉验证（K-Fold CV）
- 留一法（LOOCV）
- 分层采样（Stratified）

## 四、集成学习

| 方法 | 策略 | 代表算法 |
|------|------|----------|
| Bagging | 并行 + 平均/投票 | 随机森林 |
| Boosting | 串行 + 加权 | AdaBoost, GBDT, XGBoost |
| Stacking | 元学习器组合 | 多层模型 |

### 随机森林
- Bootstrap采样（每棵树约63.2%样本）
- 特征随机选择
- OOB误差作为无偏估计

### XGBoost 优势
- 二阶泰勒展开近似损失
- 正则化防止过拟合
- 列抽样 + 学习率缩减
- 支持缺失值自动处理
