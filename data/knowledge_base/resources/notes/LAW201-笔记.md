# 民商法基础 核心知识点笔记

## 一、民法总论

### 民事主体
- 自然人：权利能力（出生→死亡），行为能力（完全/限制/无）
- 法人：营利法人、非营利法人、特别法人
- 非法人组织：合伙企业、个人独资企业等

### 民事法律行为
- 成立要件：当事人 + 意思表示 + 标的
- 生效要件：主体适格 + 意思表示真实 + 不违法不背俗
- 效力状态：有效 / 无效 / 可撤销 / 效力待定

### 代理
- 委托代理、法定代理
- 无权代理：效力待定，被代理人追认后有效
- 表见代理：相对人有理由相信有代理权 → 有效

## 二、物权法

### 物权类型
- 所有权：占有、使用、收益、处分
- 用益物权：土地承包经营权、建设用地使用权、居住权等
- 担保物权：抵押权、质权、留置权

### 物权变动
- 不动产：登记生效
- 动产：交付生效（特殊动产登记对抗）
- 善意取得制度

## 三、合同法

### 合同订立
要约 + 承诺 = 合同成立
- 要约 vs 要约邀请
- 承诺须与要约一致（镜像规则）

### 合同效力
- 违约责任：继续履行 + 损害赔偿 + 违约金
- 免责事由：不可抗力、债权人过错
- 缔约过失责任

### 典型合同注意要点
- 买卖合同：风险转移（交付主义）
- 借款合同：利率上限（LPR四倍）
- 保证合同：一般保证 vs 连带保证

## 四、侵权责任法

### 归责原则
| 原则 | 适用情形 |
|------|----------|
| 过错责任 | 一般侵权 |
| 过错推定 | 医疗损害等 |
| 无过错责任 | 产品责任、环境污染、饲养动物等 |
| 公平责任 | 双方均无过错 |

### 特殊侵权
- 用人者责任
- 网络侵权（"通知-删除"规则）
- 安全保障义务

## 五、公司法

### 公司类型
- 有限责任公司：1-50人，人合性
- 股份有限公司：资合性，可上市
- 一人公司：特殊规制

### 公司治理
- 股东会：最高权力机关
- 董事会：经营决策机关
- 监事会：监督机关
- 三会一层制衡

### 股东权利
- 资产收益权（分红）
- 参与重大决策权（表决）
- 知情权（查账）
- 股东代表诉讼权
