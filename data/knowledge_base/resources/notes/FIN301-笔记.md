# 投资学 核心知识点笔记

## 一、资产组合理论

### 马科维茨均值-方差模型
- 期望收益：E(Rp) = ΣwᵢE(Rᵢ)
- 组合方差：σ²p = ΣΣwᵢwⱼσᵢσⱼρᵢⱼ
- 有效前沿：给定收益下风险最小的组合集合
- 最优组合：无差异曲线与有效前沿的切点

### 资本配置线（CAL）
E(Rp) = Rf + [(E(Rm)-Rf)/σm] × σp
- 斜率 = 夏普比率

## 二、CAPM 与资产定价

### CAPM 公式
E(Rᵢ) = Rf + βᵢ[E(Rm) - Rf]

### Beta 系数
βᵢ = Cov(Rᵢ, Rm)/Var(Rm)
- β > 1：进攻型股票
- β < 1：防御型股票

### 多因子模型
- Fama-French 三因子：市场 + 规模(SMB) + 价值(HML)
- Carhart 四因子：+ 动量(MOM)

## 三、期权定价

### 期权类型
- 看涨期权（Call）：买入权
- 看跌期权（Put）：卖出权
- 欧式 vs 美式（行权时间）

### 期权价值构成
期权价值 = 内在价值 + 时间价值

### 影响期权价格的因素
| 因素 | Call | Put |
|------|------|-----|
| 标的价格↑ | ↑ | ↓ |
| 行权价↑ | ↓ | ↑ |
| 波动率↑ | ↑ | ↑ |
| 到期时间↑ | ↑ | ↑ |
| 无风险利率↑ | ↑ | ↓ |

### Put-Call Parity（欧式）
C + Ke^{-rT} = P + S

## 四、行为金融

### 主要偏差
1. **过度自信**：高估自身判断准确性
2. **损失厌恶**：损失带来的痛苦 ≈ 2倍同等收益的快乐
3. **锚定效应**：过度依赖初始信息
4. **处置效应**：过早卖出盈利、过久持有亏损
5. **羊群效应**：盲目跟随市场趋势

### 前景理论（Prospect Theory）
- 价值函数在收益区凹、损失区凸
- 决策权重函数高估小概率事件

## 五、风险管理

### VaR（在险价值）
在给定置信水平下，一定持有期内的最大可能损失

### 对冲策略
- Delta对冲：对冲标的物价格风险
- 分散化：降低非系统性风险
- 止损策略：预设最大亏损线
