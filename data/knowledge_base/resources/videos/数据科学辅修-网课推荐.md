### B站推荐

## 三、实践平台与工具

### 在线练习
| 平台 | 说明 | 适合阶段 |
|------|------|----------|
| Kaggle | 数据竞赛+免费GPU | 入门→进阶 |
| 天池 | 阿里云数据竞赛 | 中级 |
| 和鲸社区 | 国内数据科学社区 | 入门→中级 |
| LeetCode | 算法练习 | 入门→进阶 |
| DataCamp | 交互式数据科学教程 | 零基础 |

### 必备工具链
- Jupyter Notebook / JupyterLab — 交互式编程环境
- VS Code + Python插件 — 开发IDE
- Git/GitHub — 代码版本管理
- Google Colab — 免费云端GPU

## 四、学习路线建议

1. **第一阶段（1-2周）**：Python基础 + Pandas/Matplotlib 入门
2. **第二阶段（3-4周）**：统计学基础 + 数据分析实战（Kaggle Titanic等）
3. **第三阶段（3-4周）**：机器学习基础算法（线性回归→决策树→SVM）
4. **第四阶段（4周+）**：Kaggle竞赛 + 个人项目实战

### 推荐书籍
- 《利用Python进行数据分析》（Wes McKinney）
- 《统计学习方法》（李航）
- 《机器学习》（周志华"西瓜书"）
- 《深度学习入门》（斋藤康毅）
